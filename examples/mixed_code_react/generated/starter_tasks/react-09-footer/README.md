# Footer

Create a footer pinned to the bottom of an 800×600 #f8fafc page. The footer is 180px high, #0f172a, and uses Arial with 36px horizontal padding. Its top row places the white 20px bold 'Acme' brand on the left and four #cbd5e1 14px links on the right with 24px gaps. A #334155 divider spans the width below. The bottom row has 12px #94a3b8 copyright text on the left and three 30px circular #1e293b social buttons on the right. Vertically distribute the two rows evenly.

Implement `src/styles.css`. You may adjust `src/App.jsx` only if needed for semantic React markup.
