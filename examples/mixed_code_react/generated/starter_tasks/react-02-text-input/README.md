# Text input

Center a 360px-wide form field on an 800×600 #f8fafc page. Stack the label, input, and helper text with 8px spacing. Use Arial. The label is 14px, weight 700, and #1e293b. The input must use border-box sizing, be 44px high, have 12px horizontal padding, a 1px #cbd5e1 border, white background, 8px radius, and 16px text. Focus changes the border to #2563eb and adds a 3px rgba(37,99,235,.18) ring. Helper text is 12px and #64748b.

Implement `src/styles.css`. You may adjust `src/App.jsx` only if needed for semantic React markup.
